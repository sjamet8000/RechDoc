### Comprendre l'interface de Liascript

Une ressource Liascript est présentée comme un diaporama interactif :

- à **gauche** : un menu escamotable listant les chapitres et sous-parties
- au **centre** : la diapositive actuelle, contenant le texte, les images, les activités, etc.
- en **haut à droite** : une série de boutons pour :

  - modifier le mode de présentation
![](assets\README\mode_pres.png)
  - modifier les couleurs de base de l'interface, la taille du texte, etc.
  - modifier la langue si une traduction est disponible
  - partager la ressource avec un QR Code
  - ouvrir l'onglet "informations" de la ressource

### Naviguer entre les sections

**Méthode 1**

- Placez votre curseur à gauche de l’écran.
- Cliquez sur le chapitre que vous souhaitez lire.
- Cliquez sur une sous-partie pour l’afficher.

**Méthode 2**

En bas à gauche de l’écran, vous pouvez cliquer :

- Flèche → pour aller à la diapositive suivante
- Flèche ← pour revenir en arrière
